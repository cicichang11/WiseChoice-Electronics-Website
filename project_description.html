Full 2022 ITP303 Final Project Summary

WiseChoice Electronics Product Listing
Cici Chang

Topic & Purpose

This website is dedicated to a theoretical electronics e-commerce company WiseChoice and serves the purpose of product listing for the company, facing both registered and non registered users, and administrators of the company. This site gives insights into electronic products’ information for the users and offers an easily manageable database for administrators.

Instructions

  • user database & credentials:

    • The user database of the site is password-protected.
    • The users and administrators need to register using username, email address, and password, where the password would be encrypted using sha256 hashing function and saved in the database.
    • After signing up an account, the users and administrators can then login using their username and password. On the back end, we would be verifying the information to decide if we can log the user in.
    • Administrators have an extra column of information called AdminNumber in the database, which can not be accessed through login or signup systems, nor can be changed. Administrators are pre-assigned roles with presetted admin numbers in the database before operating the site, for security purposes.

    • There are 3 pre-assigned admin accounts, you can use these to check my work

      • username: admin1; password: admin1; admin_num: 123
      • username: admin2; password: admin2; admin_num: 234
      • username: admin3; password: admin3; admin_num: 345

  • 3 user access level:

    • For users that are not logged in, they can access the home page (index.php), preference filter page(filter.php), the product listing page(listing.php), and detail page about each product (detail_image.php), with limitations.

      • Home page is the same as logged in users, with a button linked to the preference page.
      • Non logged in users can not set price preferences on the preference filter page.
      • Non logged in users can not view prices on the product listing page.

    • For logged in users, they can access the home page (index.php), preference filter page(filter.php), and the product listing page(listing.php), without limitations.

      • Home page is the same as non-logged in users, with a button linked to the preference page.
      • Logged in users can set all the preferences on the preferences page including price.
      • Logged in users can view the prices on the product listing page.

    • For administrators, they can access the home page (index.php), admin searching page (admin_search.php), admin product listing page(admin_listing.php), editing page(edit.php), delete page(delete_product.php), and add page(add_product.php), with some variations.

      • On the home page, administrators can only access a button that links to the admin accessible listing page, different from what users’ listing page.
      • On the admin product listing page, administrators have links to edit a product information, buttons to delete products, a button to add a product, and a button to the admin searching page to search for products using keywords in names.
      • Users don't have access to these edit, delete, and add functions.
    • The navigation bar also links to the appropriate search / filter page and product listing page, depending on whether the logged in person is a user or an admin.

  • Operations:

    • Logged-in and not logged-in users can use the site to view the information about the products that they are looking for, including brands, price, weight, conditions, shipping, price, and more details into the images on the detailed page.
    • Administrators are the only ones with the right to manage and make changes to the product the company possesses. Such as:
      • Edit
      • Add
      • Delete

Data Source

  • The product database the site relies on is modified from the Electronic Products and Pricing Data on data.world.
  • The link: https://data.world/datafiniti/electronic-products-and-pricing-data
  • Description: This is a list of over 7,000 electronic products with pricing information across 10 unique fields provided by Datafiniti's Product Database. The dataset also includes the brand, category, merchant, name, source, and more.
